#如何看消息# 
校长总结出来的媒体上发的东西大致可以分为三类

第一类就是写的是老百姓想看的东西，这部分的内容是无伤大雅，不影响上面什么的东西，就是毫不重要的，去迎合你的内容，为了有流量，所以我们就是常常看到一些特别无聊，特别没意思，特别也没关系的内容一定是流行的非常的厉害。比如说娱乐头条。

第二类叫做指令性内容，就是上面让你一定要这么说的东西，这叫宣传，就是上面想让你看见的东西。最典型的就是新闻联播，其中大部分就是想让我们看到的东西。

第三类就是各方利益集团，为了讨好某种东西，为了迎合他的目的（夹带私货），揣摩上意发出来一些东西，这些东西是为了揣摩上意，迎合上意，他想象出来的一种东西。

所以我们平时看媒体消息，要学会分辨这三类消息，抓住看消息的主要矛盾。
